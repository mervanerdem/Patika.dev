# Merge-Sort-Project

## A project for Algorithm task from Patika.dev course

### [16,21,11,8,12,22] -> Merge Sort

'Task 1' Write the stages of the array.

    [16,21,11,8,12,22]
    [16,21,11]--[8,12,22]
    [16,21]-[11]--[8,12]-[22]
    [16]-[21]-[11]--[8]-[12]-[22]
    [16,21]-[11]--[8,12]-[22]
    [11,16,21]--[8,12,22]
    [8,11,12,16,21,22]

'Task 2' What is the Big-O of this array.

    O(nlogn)
